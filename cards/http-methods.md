# Какие HTTP-методы используются в REST API и зачем?

## Ответ

HTTP-методы определяют действие над ресурсом:

| Метод  | Действие                | Идемпотентный? |
|--------|-------------------------|----------------|
| GET    | Получить данные         | Да             |
| POST   | Создать ресурс          | Нет            |
| PUT    | Заменить ресурс целиком | Да             |
| PATCH  | Частично обновить       | Нет            |
| DELETE | Удалить ресурс          | Да             |

```http
GET    /api/users      → список пользователей
POST   /api/users      → создать пользователя
PUT    /api/users/1    → заменить пользователя id=1
PATCH  /api/users/1    → обновить поля пользователя id=1
DELETE /api/users/1    → удалить пользователя id=1
```

Идемпотентный = повторный запрос даёт тот же результат.
