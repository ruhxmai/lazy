# Как называются разные стили написания имён в коде?
## Ответ
Важно знать термины на английском для работы в команде:

| Стиль | Пример | Применение |
|---|---|---|
| camelCase | `getUserName` | JS переменные, методы |
| PascalCase | `UserProfile` | Классы, React-компоненты |
| snake_case | `user_name` | Python, SQL, Ruby |
| kebab-case | `user-name` | CSS-классы, URL-слаги |
| SCREAMING_SNAKE_CASE | `MAX_RETRIES` | Глобальные константы |

**Фразы для code review:**
- *«Please use camelCase here»*
- *«This should be PascalCase — it's a component»*
- *«Constants should be SCREAMING_SNAKE_CASE»*
