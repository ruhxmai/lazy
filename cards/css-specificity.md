# Как считается специфичность (specificity) в CSS?

## Ответ

Specificity определяет, какое правило CSS «победит» при конфликте. Считается по 4 категориям, от старшей к младшей:

| Категория | Вес | Пример |
|---|---|---|
| inline-стиль | 1000 | `style="color: red"` |
| ID | 100 | `#header` |
| класс, атрибут, псевдокласс | 10 | `.btn`, `[type="text"]`, `:hover` |
| тег, псевдоэлемент | 1 | `div`, `::before` |

```css
#nav .link { color: blue; }   /* 100 + 10 = 110 */
.menu a:hover { color: red; } /* 10 + 1 + 10 = 21 */
/* Побеждает первое правило — специфичность выше */
```

`!important` перебивает всё, кроме другого `!important` с большей специфичностью — использовать только в крайнем случае.
