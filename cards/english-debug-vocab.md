# Как на английском описать процесс отладки кода?

## Ответ

Ключевые слова и фразы для отладки и обсуждения багов в команде.

```
to debug / debugging       — отлаживать / отладка
a bug / an issue           — ошибка / проблема
to reproduce a bug         — воспроизвести ошибку
a stack trace              — стек вызовов при ошибке
to set a breakpoint        — поставить точку остановки
to step through the code   — пошагово выполнять код
to log / to add logging    — добавить вывод в консоль
a root cause               — первопричина ошибки
a workaround               — временное решение
an edge case               — граничный случай
to throw / to catch        — выбросить / перехватить ошибку
flaky test                 — нестабильный тест (иногда падает)
```

**Фразы в команде:**
- "I can't reproduce it locally" — не могу воспроизвести локально
- "The root cause was a race condition" — первопричина — гонка состояний
- "Let me add some logging" — добавлю логи
- "It's a one-off issue" — разовая ошибка, не системная
- "Can you share the stack trace?" — скинь стек ошибки?
