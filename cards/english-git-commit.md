# Как правильно писать сообщения git commit на английском?

## Ответ

Используй **глагол в повелительном наклонении** (imperative mood) — как будто даёшь команду:

```
feat: add user authentication
fix: handle null value in parser
refactor: extract validation logic into helper
docs: update README with setup instructions
chore: upgrade dependencies to latest versions
test: add unit tests for CartService
```

**Типичные ошибки:**
- `added login page` — неправильно (прошедшее время)
- `adding login page` — неправильно (герундий)
- `I added login page` — неправильно (личное местоимение)

**Правильно:** `add`, `fix`, `update`, `remove`, `create`, `improve`
