# Как рассказать о своей работе на standup по-английски?
## Ответ
Стандартная структура standup: **вчера / сегодня / блокеры**.

**Вчера (Yesterday I...):**
- "...worked on / implemented / fixed / reviewed..."
- "I finished the authentication feature."
- "I reviewed two pull requests."

**Сегодня (Today I'm going to / I'll...):**
- "...continue working on / start / finish / pick up..."
- "I'll work on the payment integration."
- "I'm going to pick up the next ticket from the backlog."

**Блокеры (Blockers):**
- "No blockers on my end."
- "I'm blocked by / waiting for a review from..."
- "I need help with the deployment script."

**Полезный словарь:**
- *to pick up a ticket* — взять задачу
- *to be blocked* — быть заблокированным
- *to ship / deploy* — выкатить в продакшн
- *backlog* — список невыполненных задач
