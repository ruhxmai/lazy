# Что такое нормализация базы данных?
## Ответ
Нормализация — приведение таблиц к стандартным формам для **устранения дублирования** и аномалий при изменении данных.

- **1NF** — значения атомарны, нет повторяющихся групп столбцов
- **2NF** — соответствует 1NF + каждый неключевой атрибут зависит от **всего** первичного ключа
- **3NF** — соответствует 2NF + нет транзитивных зависимостей (неключевой → неключевой)

```sql
-- Нарушение 1NF: несколько значений в одном поле
-- phones: "111, 222" ← плохо

-- Правильно: отдельная таблица
CREATE TABLE user_phones (
  user_id INT,
  phone   VARCHAR(20)
);
```

Плюсы: меньше дублирования. Минусы: больше JOIN при чтении.
