# Что такое нормализация базы данных?
## Ответ
Нормализация — процесс организации таблиц БД для устранения дублирования данных.

- **1NF** — каждая ячейка содержит одно атомарное значение, нет повторяющихся групп столбцов
- **2NF** — выполнена 1NF + каждый неключевой столбец зависит от **всего** первичного ключа (нет частичных зависимостей)
- **3NF** — выполнена 2NF + нет транзитивных зависимостей между неключевыми столбцами

```sql
-- Нарушение 3NF: city зависит от zip, а не от id пользователя
-- Исправление: выносим zip → city в отдельную таблицу
CREATE TABLE zip_codes (
  zip  VARCHAR(10) PRIMARY KEY,
  city VARCHAR(100) NOT NULL
);
```
