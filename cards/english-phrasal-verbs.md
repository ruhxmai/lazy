# Какие фразовые глаголы чаще всего используют разработчики на созвонах?

## Ответ

```
set up       — настроить            "I'll set up the environment"
roll out     — выкатить             "We rolled out the new feature"
roll back    — откатить             "Let's roll back this deploy"
break down   — разбить на части     "Break down the task into subtasks"
figure out   — разобраться          "I need to figure out why it fails"
look into    — изучить              "I'll look into the bug"
come up with — придумать            "She came up with a workaround"
run into     — столкнуться          "I ran into an issue with auth"
check in     — проверить/отметиться "Let's check in tomorrow"
follow up    — уточнить позже       "I'll follow up after the meeting"
```

- Фразовые глаголы часто нельзя перевести дословно — их значение нужно запоминать целиком
- В письменной документации чаще используют более формальные аналоги: `investigate` вместо `look into`, `resolve` вместо `figure out`
