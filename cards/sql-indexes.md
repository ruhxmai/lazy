# Что такое индекс в SQL и когда его использовать?

## Ответ

Индекс — вспомогательная структура данных (обычно B-Tree), которая ускоряет поиск строк по колонке без полного сканирования таблицы.

**Когда создавать:**
- Колонка часто используется в `WHERE`, `JOIN ON`, `ORDER BY`
- Таблица большая (от нескольких тысяч строк)

**Когда не стоит:**
- Таблица маленькая — оптимизатор и так быстро сканирует
- Колонка часто обновляется — каждый `INSERT`/`UPDATE` перестраивает индекс

```sql
CREATE INDEX idx_users_email ON users(email);

-- теперь быстро:
SELECT * FROM users WHERE email = 'a@b.com';
```
