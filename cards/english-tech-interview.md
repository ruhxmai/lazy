# Как рассказывать об опыте по-английски на техническом собеседовании?

## Ответ

Ключевые фразы для технического интервью:

```
Опыт работы:
  I have X years of experience in...
  I've been working with React for about 2 years.
  I'm familiar with / I have hands-on experience with...

Описание проекта:
  I worked on a project where...
  My responsibility was to...
  I implemented / built / developed / optimized...

Технические ответы:
  To put it simply...       (Если говорить просто...)
  The key difference is...  (Ключевое отличие...)
  It depends on...          (Это зависит от...)
  Under the hood...         (Под капотом...)
  The trade-off here is...  (Компромисс здесь...)

Когда не знаешь ответа:
  I'm not 100% sure, but I think...
  I'd need to look that up to be certain.
  I haven't worked with that directly, but I'd approach it by...
```

- При ступоре не молчи — объясни ход мыслей: *"Let me think through this..."*
- Говори о трейдоффах: *"The advantage is X, but the downside is Y"*
