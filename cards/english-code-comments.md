# Как правильно писать комментарии к коду на английском?

## Ответ

Комментарии пишутся в **повелительном наклонении** (imperative mood) — глагол первым, без артикля перед ним:

```js
// Calculate total price including tax
// Fetch user profile from API
// Return early if input is invalid
// Convert timestamp to human-readable date
```

Специальные теги:

```js
// TODO: add pagination support
// FIXME: crashes when list is empty
// NOTE: intentional for backwards compatibility
// HACK: workaround for Safari rendering bug
```

Не описывай **что** — описывай **зачем**.
