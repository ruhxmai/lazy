# Чем псевдокласс отличается от псевдоэлемента в CSS?

## Ответ

- **Псевдокласс** (`:hover`, `:nth-child()`, `:focus`, `:first-child`) выбирает элемент в особом **состоянии** или позиции — одно двоеточие `:`.
- **Псевдоэлемент** (`::before`, `::after`, `::first-line`) создаёт/выбирает **часть** элемента, которой нет в DOM — два двоеточия `::`.

```css
/* Псевдокласс: стиль при наведении */
.button:hover {
  background: darkblue;
}

/* Псевдокласс: каждый второй элемент списка */
li:nth-child(2n) {
  background: #f5f5f5;
}

/* Псевдоэлемент: добавляет контент перед элементом */
.required::before {
  content: "*";
  color: red;
}
```

- `::before` и `::after` требуют свойство `content`, иначе не отобразятся
- Старый синтаксис с одним `:` для псевдоэлементов (`:before`) тоже работает — для обратной совместимости, но стандарт — `::before`
