# Как читать техническую документацию на английском?
## Ответ
Ключевые слова и фразы в документации:

| Фраза | Перевод |
|---|---|
| **Deprecated** | устаревшее, не рекомендуется |
| **Returns** | что возвращает функция |
| **Throws** | какую ошибку выбрасывает |
| **Optional** | необязательный параметр |
| **Defaults to** | по умолчанию равно |
| **See also** | смотри также |
| **Note / Caution / Warning** | заметка / предупреждение |
| **Under the hood** | под капотом (как работает внутри) |

**Стратегия:** сигнатура функции → описание → параметры → примеры кода. Примеры важнее текста.
