# Какие английские слова и фразы важны при чтении технической документации?
## Ответ
| Фраза | Перевод |
|---|---|
| **returns** | возвращает (о функции) |
| **deprecated** | устаревшее, не рекомендуется |
| **optional** | необязательный параметр |
| **defaults to X** | по умолчанию равен X |
| **throws** | выбрасывает исключение |
| **override** | переопределить |
| **under the hood** | внутри, «под капотом» |
| **fallback** | запасной вариант |
| **callback** | функция обратного вызова |
| **boilerplate** | шаблонный повторяющийся код |
