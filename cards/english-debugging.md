# Как говорить о дебаггинге на английском?

## Ответ

| Русский | English |
|---------|---------|
| Я дебажу | I'm debugging / stepping through the code |
| Найти баг | To track down / reproduce a bug |
| Точка останова | Breakpoint |
| Стек вызовов | Call stack |
| Падает / крашится | It crashes / throws an error |
| Логировать | To log / to console.log |
| Что-то пошло не так | Something went wrong / It's broken |

**Примеры фраз:**
- *"I set a breakpoint and stepped through the function."*
- *"The bug only reproduces in production."*
- *"I checked the call stack and found the issue."*
- *"Can you help me figure out why this is failing?"*
