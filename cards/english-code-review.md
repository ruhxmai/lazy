# Как писать комментарии на code review по-английски?
## Ответ
Полезные фразы и аббревиатуры:

| Фраза | Смысл |
|-------|-------|
| `LGTM` (Looks Good To Me) | Одобряю |
| `nit:` | Незначительное замечание |
| `Could you explain why...?` | Можешь объяснить почему...? |
| `This might cause...` | Это может вызвать... |
| `Consider using...` | Рассмотри использование... |
| `I'd suggest...` | Я бы предложил... |
| `Is this intentional?` | Это намеренно? |
| `Please address before merge` | Исправь до мержа |
| `Blocking` / `Non-blocking` | Блокирует / не блокирует мерж |

**Тон:** пиши вопросами (`Could we...?`) вместо приказов (`Change this`).
