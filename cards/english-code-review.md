# Какие фразы используют при code review по-английски?
## Ответ
**Предложить изменение (вежливо):**
- *Could we consider using X instead?* — Может, рассмотрим X?
- *What do you think about...?* — Как ты смотришь на...?
- *This might be cleaner if...* — Это может быть чище, если...

**Попросить пояснение:**
- *Could you explain the reasoning here?* — Можешь объяснить логику?
- *I'm not sure I follow — what does X do?* — Не совсем понимаю — что делает X?

**Ответить на замечание:**
- *Good catch, I'll fix it.* — Хорошо замечено, исправлю.
- *Makes sense, updating now.* — Логично, сейчас обновлю.
- *I'll address this in a follow-up PR.* — Исправлю в следующем PR.
