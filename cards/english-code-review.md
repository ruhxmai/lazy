# Какие английские фразы используются при code review?

## Ответ

Типичные комментарии в ревью:

| Фраза | Значение |
|-------|----------|
| **LGTM** (Looks Good To Me) | Всё ок, одобряю |
| **nit:** | Мелкое замечание, необязательно |
| **blocker:** | Критичное — нужно исправить |
| **Could you rename this to...?** | Предложение переименовать |
| **This might cause issues if...** | Может сломаться, если... |
| **Consider using...** | Предлагаю рассмотреть... |
| **Let's extract this into...** | Давай вынесем это в... |
| **This is out of scope** | Это за рамками задачи |
| **Can you add a test for this?** | Добавь тест для этого |
