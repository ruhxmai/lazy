# Какие фразы использует разработчик при code review на английском?
## Ответ
**Оставить комментарий:**
- `LGTM` — Looks Good To Me (одобряю)
- `Nit:` — незначительное замечание, можно проигнорировать
- "Could you add a comment explaining why this is done?"
- "This might cause a bug when X happens."
- "Optional: you could simplify this with..."
- "Blocking: this needs to be fixed before merge."

**Ответить на review:**
- "Good catch, fixed!"
- "You're right, I'll refactor this."
- "Done in the latest commit."
- "I disagree because..., but happy to change it if needed."

**Попросить review:**
- "Ready for review."
- "Could you take a look when you have a moment?"
- "I've addressed all your comments."
