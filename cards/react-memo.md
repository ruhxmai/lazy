# Зачем нужен React.memo и когда его применять?

## Ответ

`React.memo` — HOC, который мемоизирует результат рендера компонента. Повторный рендер пропускается, если переданные пропсы не изменились (поверхностное сравнение).

```jsx
const Button = React.memo(({ onClick, label }) => {
  console.log('render');
  return <button onClick={onClick}>{label}</button>;
});
```

Используй вместе с `useCallback` для стабилизации функций-пропсов. Не оборачивай все компоненты подряд — мемоизация сама стоит памяти и времени на сравнение. Применяй там, где рендер дорогой или компонент рендерится очень часто.
