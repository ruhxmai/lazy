# Что такое React.memo и чем он отличается от useMemo?
## Ответ
`React.memo` — HOC, который предотвращает ре-рендер **компонента**, если его props не изменились.

```jsx
const UserCard = React.memo(({ name, age }) => {
  return <div>{name}, {age}</div>
})

// Компонент не ре-рендерится, если name и age не изменились
```

**Сравнение:**
| | React.memo | useMemo |
|---|---|---|
| Что мемоизирует | Компонент (HOC) | Вычисленное значение |
| Где используется | Обёртка компонента | Внутри компонента |

Используй `React.memo`, когда компонент часто получает одинаковые props, а рендер дорогой.
