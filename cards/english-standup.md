# Как говорить на daily standup по-английски?

## Ответ

Standup состоит из трёх блоков:

**Yesterday / Last time (что сделал):**
- *"Yesterday I worked on the login page."*
- *"I finished implementing the user authentication."*
- *"I fixed the bug in the payment flow."*

**Today (что планирую):**
- *"Today I'm going to work on the dashboard."*
- *"I'll be reviewing the open PRs."*
- *"I'm planning to write tests for the API."*

**Blockers (что мешает):**
- *"No blockers."*
- *"I'm blocked by the missing API documentation."*
- *"I need a review on my PR before I can continue."*

**Полезные фразы:**
- *"I'm in the middle of..."* — в процессе
- *"It's almost done"* — почти готово
- *"I'm waiting for feedback"* — жду обратной связи
- *"It took longer than expected"* — заняло больше времени
