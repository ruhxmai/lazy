# Как провести daily standup на английском?
## Ответ
На стендапе отвечают на 3 вопроса:

**1. What did you do yesterday?**
> "Yesterday I worked on the login form and fixed a bug in the validation logic."

**2. What are you going to do today?**
> "Today I'm going to implement the API integration and write unit tests."

**3. Any blockers?**
> "I'm blocked on the design mockups — waiting for the designer."
> "No blockers."

**Полезные фразы:**
- *I'm still working on...* — ещё работаю над...
- *I need to sync with...* — нужно поговорить с...
- *I'm about to start...* — сейчас начну...
- *It's almost done* — почти готово
- *I just finished...* — только что закончил...
