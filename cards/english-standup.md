# Как провести ежедневный стендап на английском?

## Ответ

**Что сделал вчера:**
- "Yesterday I worked on / finished / fixed…"
- "I implemented the login form and submitted a PR."

**Что планирую сегодня:**
- "Today I'm going to / I'll be working on…"
- "I'll continue with the user profile page."

**Блокеры:**
- "I'm blocked by…" / "No blockers."
- "I need help with the API integration."
- "I'm waiting for a review from the backend team."

Говори конкретно и кратко — стендап обычно 15 минут на всю команду.
