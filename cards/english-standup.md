# Как говорить на standup по-английски?

## Ответ

Стандартная трёхчастная структура:

```
Yesterday I worked on [task].
Today I'm going to [task].
I'm blocked by [issue]. / No blockers.
```

**Полезные фразы:**
- *I'm currently working on...* — сейчас работаю над...
- *I need help with...* — мне нужна помощь с...
- *It's almost done / in review* — почти готово / на ревью
- *I'm waiting for feedback* — жду обратной связи
- *I'll pick up [task] next* — следующей возьму задачу...
- *It's blocked on the backend side* — блокируется на стороне бэкенда
