# Как рассказать о своей работе на daily standup по-английски?

## Ответ

Standup — три вопроса: что сделал, что буду делать, есть ли блокеры.

**Yesterday / What I did:**
```
Yesterday I worked on the login form validation.
I fixed a bug with the dropdown menu.
I reviewed two pull requests.
```

**Today / What I'm doing:**
```
Today I'm going to implement the user profile page.
I'm planning to write unit tests for the auth module.
I'll be working on the API integration.
```

**Blockers:**
```
No blockers.
I'm blocked by — I'm waiting for the design mockups.
I need help with — I'm stuck on the CORS issue.
```

Полезные фразы:
- *I'm almost done with...* — почти закончил...
- *It took longer than expected* — заняло больше времени, чем планировал
- *I'll pick up where I left off* — продолжу с того места, где остановился
