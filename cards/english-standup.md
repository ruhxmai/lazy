# Как провести Daily Standup на английском?

## Ответ

Daily standup — короткая встреча (≤15 мин). Три стандартных вопроса:

**1. What did you do yesterday?**
> "Yesterday I worked on the login form and fixed a bug in the auth flow."

**2. What will you do today?**
> "Today I'm going to implement the password reset feature."

**3. Any blockers?**
> "I'm blocked on the API spec — waiting for the backend team."
> "No blockers, moving forward."

**Полезные фразы:**
- *"I'm still working on..."* — ещё не закончил
- *"I wrapped up..."* — завершил задачу
- *"I'm about to start..."* — скоро начну
- *"Could someone help me with...?"* — нужна помощь
- *"I'll pick up..."* — возьму задачу
