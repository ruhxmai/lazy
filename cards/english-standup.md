# Как говорить на daily standup по-английски?

## Ответ

Стандартная структура: **вчера / сегодня / блокеры**

```
// Yesterday (что сделал вчера)
"Yesterday I worked on the login feature."
"I fixed a bug with the API timeout."
"I reviewed two pull requests."

// Today (что планируешь сегодня)
"Today I'm going to implement the user profile page."
"I plan to write unit tests for the auth module."

// Blockers (что мешает)
"I'm blocked on the design mockups — waiting for the designer."
"No blockers." / "I'm good." / "Nothing blocking me."

// Другие полезные фразы
"I need help with..."              // мне нужна помощь с...
"I'll pair with [name] on..."      // буду работать в паре с...
"It should be ready by EOD."       // к концу дня (End Of Day)
"It's almost done, ~80% complete." // почти готово
```
