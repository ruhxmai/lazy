# Чем Map и Set отличаются от объекта и массива?
## Ответ
`Map` — коллекция пар «ключ → значение», где ключ может быть любого типа.  
`Set` — коллекция уникальных значений.

```js
const map = new Map();
map.set('name', 'Alice');
map.set(42, 'число как ключ');
map.get('name'); // 'Alice'
map.size;        // 2

const set = new Set([1, 2, 2, 3, 3]);
set.size;    // 3 — дубликаты удалены
set.has(2);  // true
[...set];    // [1, 2, 3]
```

- `Map` сохраняет порядок вставки; ключ может быть объектом
- `Set` идеален для хранения уникальных значений и быстрой проверки наличия
- `Object` — ключи только строки/символы; `Array` позволяет дубликаты
