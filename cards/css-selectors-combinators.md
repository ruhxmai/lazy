# Что означают комбинаторы (пробел), `>`, `+` и `~` в CSS-селекторах?

## Ответ

Комбинаторы задают связь между селекторами через **отношение элементов в DOM**:

| Комбинатор | Название | Что выбирает |
|---|---|---|
| `A B` (пробел) | потомок (descendant) | любой `B` внутри `A`, на любой глубине |
| `A > B` | прямой потомок (child) | только `B`, который **прямой** ребёнок `A` |
| `A + B` | соседний брат (adjacent sibling) | `B`, идущий **сразу** после `A` на том же уровне |
| `A ~ B` | братья (general sibling) | **любой** `B` после `A` на том же уровне |

```html
<div class="card">
  <p>Первый параграф</p>
  <span>
    <p>Вложенный параграф</p>
  </span>
  <p>Второй параграф</p>
</div>
```

```css
.card p { }     /* оба <p> и вложенный — потомки на любой глубине */
.card > p { }   /* только два прямых <p>, вложенный в span — не выбирается */
.card + p { }   /* ничего — нет <p>, идущего сразу за .card на том же уровне */
p + p { }       /* второй параграф — идёт сразу после первого */
p ~ p { }       /* второй параграф — любой <p>-брат после первого */
```

**Мнемоника:** `>` — «ровно один шаг вниз», `+` — «ровно один шаг вправо», `~` — «сколько угодно шагов вправо».
