# Чем отличаются px, %, em, rem и vh/vw в CSS?

## Ответ

**`px`** — абсолютная единица, фиксированный размер в пикселях, не зависит от контекста.

**`%`** — относительно размера родительского элемента.

**`em`** — относительно `font-size` **текущего** элемента (а для `font-size` самого элемента — относительно родителя). Из-за этого `em` может накапливаться при вложенности.

**`rem`** — относительно `font-size` корневого элемента `<html>` (обычно 16px). Не зависит от вложенности — предсказуемее, чем `em`.

**`vh` / `vw`** — 1% от высоты/ширины **окна просмотра** (viewport).

```css
html { font-size: 16px; }

.parent {
  font-size: 20px;
  padding: 2em;     /* 2 * 20px = 40px — от родителя */
}

.child {
  font-size: 1.5em; /* 1.5 * 20px = 30px (родительский font-size) */
  padding: 2rem;     /* 2 * 16px = 32px — всегда от html, вложенность не важна */
}

.hero {
  height: 100vh;     /* всегда высота экрана, даже без контента */
  width: 50vw;       /* половина ширины окна */
}
```

**Практика:** для `font-size` чаще используют `rem` (предсказуемо и легко масштабировать через `html { font-size }`), для отступов внутри компонента — `em` (масштабируется вместе с текстом компонента), для полноэкранных блоков — `vh`/`vw`.
