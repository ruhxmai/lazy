# Что такое IIFE (Immediately Invoked Function Expression)?

## Ответ

Функция, которая **вызывается сразу после объявления**. Круглые скобки вокруг неё нужны, чтобы движок распознал в этом месте выражение (expression), а не декларацию функции.

```js
(function () {
  console.log("Выполнилась сразу");
})();

// Со стрелочной функцией
(() => {
  const secret = 42;
  console.log(secret);
})();
```

Используется, чтобы **не засорять глобальную область видимости** — переменные внутри IIFE недоступны снаружи. До появления `let`/`const` и ES-модулей это был основной способ изолировать код.
