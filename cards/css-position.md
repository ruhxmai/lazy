# Какие значения у CSS-свойства position и чем они отличаются?
## Ответ
- `static` — по умолчанию, элемент в обычном потоке документа  
- `relative` — смещается относительно своего обычного места, остаётся в потоке  
- `absolute` — выпадает из потока, позиционируется относительно ближайшего предка с non-static позицией  
- `fixed` — фиксируется относительно вьюпорта, не прокручивается  
- `sticky` — ведёт себя как `relative` до достижения порога прокрутки, затем «прилипает»

```css
.parent { position: relative; }
.child  { position: absolute; top: 0; right: 0; } /* угол родителя */
.header { position: sticky; top: 0; }             /* прилипает при скролле */
```
