# Чем псевдоэлементы (`::before`, `::after`) отличаются от псевдоклассов (`:hover`, `:nth-child`)?

## Ответ

**Псевдокласс** (`:hover`, `:focus`, `:nth-child`) выбирает уже существующий элемент в определённом **состоянии** или позиции. Он не создаёт ничего нового в DOM.

**Псевдоэлемент** (`::before`, `::after`, `::first-line`) создаёт **виртуальный подэлемент**, которого нет в реальном DOM — он существует только визуально, через CSS.

```css
/* Псевдокласс — состояние существующего элемента */
button:hover {
  background: blue;
}

/* Псевдоэлемент — виртуальный элемент, требует content */
button::after {
  content: "→";
  margin-left: 4px;
}
```

Ключевое правило: `::before` и `::after` **обязаны** иметь свойство `content`, иначе не отрендерятся. Двойное двоеточие `::` — современный синтаксис для псевдоэлементов, одинарное `:` — для псевдоклассов (хотя `:before` тоже работает для обратной совместимости).
