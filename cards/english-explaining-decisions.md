# Как на английском объяснить техническое решение или trade-off?

## Ответ

Озвучить выбор:
```
We decided to go with option A because it's easier to maintain.
I went with a simpler approach here to save time.
```

Объяснить компромисс (trade-off):
```
The trade-off here is speed versus readability.
This approach is faster, but it's harder to test.
It works, but it's not the most scalable solution.
```

Сравнить варианты:
```
Option A is more performant, but option B is easier to extend later.
I'd recommend X over Y since our team is already familiar with it.
```

Признать ограничение:
```
This is a temporary solution — we'll need to revisit it once traffic grows.
There's a known limitation: it doesn't handle concurrent updates yet.
```

Пригодится в код-ревью, на дизайн-обсуждениях и на собеседованиях, когда просят объяснить свой выбор.
