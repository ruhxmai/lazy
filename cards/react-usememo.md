# Для чего используется хук useMemo в React?
## Ответ
`useMemo` мемоизирует результат вычисления — пересчёт происходит только при изменении зависимостей. Помогает избежать дорогих вычислений при каждом рендере.

```jsx
const sortedItems = useMemo(() => {
  return [...items].sort((a, b) => a.price - b.price);
}, [items]); // пересчитается только при изменении items
```

- **Отличие от `useCallback`**: `useMemo` кэширует **значение**, `useCallback` — **функцию**
- Не используй для каждого вычисления: мемоизация сама имеет стоимость
- Применяй когда: вычисление медленное, или результат передаётся в `React.memo`-компонент
