# Для чего нужны useMemo и useCallback?
## Ответ
`useMemo` кэширует **результат вычисления** и пересчитывает его только при изменении зависимостей — используется для «дорогих» операций.  
`useCallback` кэширует **саму функцию**, чтобы не создавать её заново при каждом рендере — полезно при передаче коллбэков в дочерние компоненты.

```js
// Пересчитывается только при изменении a или b
const result = useMemo(() => heavyCompute(a, b), [a, b]);

// Создаётся заново только при изменении id
const handleClick = useCallback(() => {
  doSomething(id);
}, [id]);
```
