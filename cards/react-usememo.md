# Для чего нужен хук useMemo в React?
## Ответ
`useMemo` кэширует результат вычисления и пересчитывает его только при изменении зависимостей. Используется для дорогих операций, чтобы избежать лишних вычислений при каждом рендере.

```js
const sortedList = useMemo(() => {
  return items.sort((a, b) => a.price - b.price);
}, [items]);
```

Без `useMemo` сортировка запускалась бы при каждом рендере компонента. Не стоит использовать его для простых вычислений — это лишние накладные расходы.
